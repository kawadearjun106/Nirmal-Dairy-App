# 🥛 Nirmal Dairy App

A Kotlin-based Android application developed for managing dairy operations efficiently using Firebase services for cloud storage and real-time data management. The application provides a simple and user-friendly interface for handling dairy-related records and operations.

---

# 🚀 Features

* Dairy management system
* Firebase-based cloud storage
* User-friendly mobile interface
* Real-time data management
* Efficient record handling
* Clean and simple UI

---

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/kawadearjun106/Nirmal-Dairy-App.git
```

---

## 2️⃣ Open in Android Studio

* Open Android Studio
* Select "Open Project"
* Choose the cloned repository folder

---

## 3️⃣ Firebase Setup

1. Create a Firebase project
2. Connect the Android app to Firebase
3. Download `google-services.json`
4. Place it inside:

```bash
app/
```

---

## 4️⃣ Sync Gradle

Allow Android Studio to download and sync all dependencies.

---

## 5️⃣ Run the Application

* Connect an Android device or start an emulator
* Click the ▶️ Run button in Android Studio
